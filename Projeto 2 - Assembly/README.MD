# PROJETO 2 - IMPLEMENTAÇÃO DE ALGORITMO EM MIPS ASSEMBLY
### COMP0415 - ARQUITETURA DE COMPUTADORES - T03

Este trabalho tem como objetivo a implementação de um código em MIPS Assembly.

Para explorar melhor as capacidades da ISA, foi escolhido como algoritmo a busca binária, capaz de encontrar itens em vetores ordenados em tempo $O(log \; n)$.

Para a implementação, foram utilizadas diversas funções, além da própria busca; o armazenamento de valroes em vetores; chamadas de sistema para interação com o usuário; e recursividade.

O arquivo "**Busca Binária Recursiva - Assembly.pdf**" contém a apresentação e o detalhamento do algoritmo e sua implementação, contendo imagens do datapath de determinadas instruções executadas ao longo do algoritmo. Já o **binarySearchRecursive.asm** possui o código utilizado na apresentação. 

Os outros arquivos presentes nesta pasta foram utilizados para iterar entre versões do projeto e planejar novas etapas deste, sendo complementos dos arquivos principais.
